    AR Applications

    The first AR project was completed utilising the Unity Engine and its AR, XR, and Vuforia SDK plugins. It's an AR-based game that makes use of the camera on gadgets like mobile phones and webcams. This game is designed for Android smartphones. To play, simply position the camera on the device and press the start button. Balloons will then keep dropping from the top, and you'll need to shoot them using the UI shot button on the bottom right.

  • Vuforia SDKs were mostly used with Image targets, and work was done with single-level, multiple-level, and other ITs.

  • Vuforia was used in numerous projects, including the solar system project, the piano, single image target, and multiple image target.
